## About The Project
This is a classic Pong clone developed to demonstrate core game development principles using Unity and C#. The project focuses on creating a smooth, physics-based arcade experience with a polished game loop.

## Key Features
* **Physics-Based Gameplay:** Real-time collision detection and ball physics.
* **Responsive Controls:** Precise paddle movement for a classic arcade feel.
* **Dynamic Scoring:** UI system that tracks player progress.

## Technical Details
* **Engine:** Unity
* **Language:** C#
* **Assets:** Custom sprites and UI implemented via TextMeshPro.
* **Architecture:** Clean code structure utilizing specialized managers for game state and audio.

## How to Play
* **Player 1:** Use 'W' and 'S' keys to move.
* **Player 2:** Use 'Up' and 'Down' arrow keys to move.
* **Goal:** Deflect the ball past your opponent to score points!
